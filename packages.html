Packages

Release packages for Windows x64, Linux x64, and macOS ARM.

GlowFlow ships as the `glow` CLI binary. The packaging flow builds a standalone archive for each target platform so end users do not need Rust installed just to run GlowFlow.

Windows x64

x86_64-pc-windows-gnu

Archive format: `zip`

Linux x64

x86_64-unknown-linux-gnu

Archive format: `tar.gz`

macOS ARM

aarch64-apple-darwin

Archive format: `pkg`
